# 🌿 New Plants Website

A responsive React-based web application showcasing a curated selection of new plants. The site fetches plant data from a public API and displays it in a clean, modern grid layout.

## 🚀 Features

- Fetches plant data from a remote API
- Displays up to **6 latest plants**
- Responsive grid layout (1–3 columns)
- Tailwind CSS for elegant styling
- React functional components with hooks
